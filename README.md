# API Silex Skeleton

## Install

With Docker and Composer installed in your system just run the following
command on project root:

```bash
./bin/minion

3) Start

```

## Usage

### Minion (shell task manager)

Run `./bin/minion` and you will see this options:

1) Help

2) PHP Docs

3) Start

4) Close

5) Restart

6) Quit
